# Student Academic Performance Prediction using Machine Learning

## Overview
This project predicts student academic performance using machine learning models and provides insights to identify at-risk students.

## Dataset
- Source: UCI Machine Learning Repository
- Size: 395 records × 33 features

## Models Used
- Linear Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest achieved the highest R² score (~0.81).

## Outcome
The system helps teachers identify students who may require early academic support.
